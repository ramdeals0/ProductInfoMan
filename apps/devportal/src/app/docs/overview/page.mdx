# Overview

ProductInfoMan is a high-performance ecommerce **Product Information Management (PIM)** platform. It provides APIs for catalog authoring, taxonomy, imports, workflow, search projection, publishing, audit, and eventing.

## Base URLs

<EnvironmentTable />

Health check (no auth): `GET /health` on the API host root.

## API domains

- **Products & Variants** — create, update, attribute values, variant trees
- **Taxonomy & Facets** — categories, attributes, facet definitions and rules
- **Imports** — CSV upload, validation, commit
- **Workflow** — submit, approve, reject, publish transitions
- **Search** — index management and customer-facing search/facets
- **Publishing** — channels, field mappings, export jobs
- **Audit & Reporting** — change logs and operational dashboards
- **Events** — outbox, dead-letter queue, replay
- **Storefront catalog** — read-only search, categories, and product detail (same API, org-scoped)
